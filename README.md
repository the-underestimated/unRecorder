# UnRecorded by Gz.

Script untuk rekapitulasi data Unrecord dari Stock Opname, dihost menggunakan Streamlit.

## Cara Penggunaan Script

1. Export chat grup Stock Opname dengan klik Titik Tiga di pojok atas kanan, kemudian pilih _More_, kemudian pilih _Export Chat_. Saat ditanya apakah perlu _include media_, pilih _Include Media_.
2. Simpan hasil export tersebut pada penyimpanan perangkat.
3. Buka situs https://unrecorder-gz.streamlit.app/
4. Upload file hasil export chat tersebut di web.
5. Pilih tanggal awal dan akhir Stock Opname untuk membatasi pengambilan data chat.
6. Pilih bahasa yang digunakan pada WhatsApp (_default_-nya sama dengan bahasa pada sistem _smartphone_)
7. Pilih format waktu (24 jam atau 12 jam)
8. Klik tombol 'Olah Data!'
9. Setelah pengolahan selesai, maka dapat diklik tombol 'Download Data Unrecord'
